# Projet IoT : Contrôleur de Musique Connecté

Ce projet est un **système IoT pour contrôler et visualiser la musique** à l'aide de composants électroniques comme un buzzer, des LEDs et un bouton. Il permet également de suivre les chansons et de voir quelle musique est en train de jouer via une interface.

---

## Fonctionnalités

- **Buzzer** : Joue un son lorsque l'appareil est activé ou pour notifier certaines actions.
- **LEDs** :
  - **Rouge** : Indique que le système est éteint.
  - **Vert** : Indique que le système est activé.
- **Bouton** : Permet de passer à la chanson suivante.
- **Interface utilisateur** : 
  - Affiche la chanson en cours.
  - Permet de passer à la chanson suivante directement depuis l'interface.

---

## Matériel utilisé

- Microcontrôleur
- Buzzer
- LED rouge et LED verte
- Bouton
- Câbles
